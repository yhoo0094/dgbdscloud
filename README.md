# DGBDS CLOUD

## 프로젝트 목적

* 클라우드(AWS) 관련 사내 강연을 진행하기에 앞서 사전 조사를 진행하기 위한 웹 페이지를 구축한다.
* 프로젝트를 클라우드로 배포하여 사전 조사와 동시에 해당 강연을 통해 배울 수 있는 내용을 홍보한다.
* 배포주소: https://dgbdscloud.click/
